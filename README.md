# C Recursion Programs

This repository contains C programs that demonstrate fundamental recursion concepts and problem-solving techniques. These examples help in understanding recursive function calls, mathematical recursion, and path-counting algorithms.

## Programs Included

1. **powerrectangle.c**
   - Calculates the power of a number using simple recursion.

2. **fibonacci.c**
   - Generates the Fibonacci sequence using recursion.

3. **stairpathcount.c**
   - Counts the total number of ways to reach the top of a staircase using recursion.

4. **powerlogrec.c**
   - Calculates the power of a number using logarithmic recursion (Exponentiation by Squaring).

5. **mazepath.c**
   - Finds the total number of possible paths in a maze using recursion.

## Concepts Covered

- Recursion
- Base Case
- Recursive Calls

- Maze Path Counting
- Logarithmic Power Algorithm
